# Hotel Management System

A **Hotel Management System** built using **Java Swing** and **MySQL** to manage hotel operations such as room handling, customer management, bookings, and revenue tracking. This project provides a simple and user-friendly interface suitable for academic and internship purposes.

##  Features

- **Dashboard Overview**
  - Displays total rooms, customers, booking count, and total revenue
- **Room Management**
  - Add, update, remove rooms and manage room status
- **Customer Management**
  - Store and manage customer personal details
- **Booking Management**
  - Handle room booking, check-in, check-out, and calculate cost
- **User Login System**
  - Secure authentication with username and password
- **MySQL Database Integration**
  - Smooth data storage and retrieval
